# TXT Reader - 摸鱼文本阅读器 v0.0.1

Windows 下的轻量级 TXT 文本阅读器，专为阅读超大文本文件（小说、日志等）设计。

## 功能特性

- **虚拟渲染**：只绘制屏幕上可见的几十行，支持数百万字符、数十万行的文件
- **进度保存**：支持多个命名存档点，随时保存/恢复阅读进度
- **自动恢复**：关闭时自动保存位置，下次打开同一文件自动恢复
- **透明背景**：默认半透明（桌面透视），鼠标移入后显示完整界面
- **字体颜色自定义**：可自由选择文字颜色
- **内存友好**：仅缓存行索引（10万行 ≈ 800KB），不将完整文件加载到内存

## 快捷键

| 按键 | 功能 |
|------|------|
| `Ctrl+O` | 打开文件 |
| `Ctrl+S` | 保存进度 |
| `Ctrl+L` | 选择已保存的进度 |
| `Ctrl+Q` | 退出程序 |
| `Esc` | 最小化窗口 |
| `↑` `↓` | 逐行滚动 |
| `PgUp` `PgDn` | 翻页 |
| `Home` `End` | 跳到文件开头/结尾 |
| 鼠标滚轮 | 滚动 |
| 拖拽右侧滑块 | 快速跳转 |

## 使用说明

### 直接运行

需要 Python 3.7+，无需安装任何第三方库：

```bash
python main.py
```

### 打包为 EXE

```bash
pip install pyinstaller
pyinstaller -F -w main.py -n TXTReader
```

生成的 `dist/TXTReader.exe` 可直接双击运行。

## 进度追踪方案

采用**字节偏移量（Byte Offset）**记录阅读进度，定位精度达字节级别，不受编码和文件修改影响。
